# HUI-Tools

HUI Tools is a small suite of software utilities for the Mackie HUI (Human User Interface) studio controller. It was created to solve a specific problem: the vacuum-fluorescent display (VFD) on a HUI that has failed, leaving the user unable to see plug-in parameters, dialog boxes, or other text that Pro Tools sends to that screen. HUI Tools intercepts and re-displays that information on a computer monitor, and also provides a comprehensive hardware test tool for diagnosing HUI problems.

The suite consists of two programs:
* **HUI-Display** — Intercepts the MIDI data that Pro Tools sends to the HUI's main display, shows it in a floating on-screen window, and passes all traffic through to the hardware unchanged so the rest of the unit continues to work normally.
* **HUI-Test** — Connects directly to the HUI hardware and lets you individually test every feature of the unit: Buttons, LEDs, faders, V-pot rings, VU meters, the VFD display, relays, and the click and beep circuits.

Learn how to install the program in installation.md. See the instruction manual for complete details on operation of each program.
